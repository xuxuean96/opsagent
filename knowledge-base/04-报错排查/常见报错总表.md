# 常见报错总表

| 现象 / 关键词 | 优先怀疑 | 第一轮检查 |
|---|---|---|
| 加载失败 | 环境 / 注册 / 位数 | 目录、注册、依赖 |
| 打不开文件 | 路径 / 权限 | 文件是否存在、权限是否够 |
| 输出为空 | 配置 / 映射 | XML、字段映射、输入数据 |
| 结果不对 | 规则 / 版本 | 版本、模式、映射规则 |
| 日志没有生成 | 路径 / 权限 | 日志目录是否可写 |

## 排障原则

先问“是什么现象”，再问“用了哪个组件”，最后问“日志里第一条错误是什么”。

# 加密卡客户端调用

## Java 客户端常用方式

```bash
java -jar KmEncryptCardClient.jar -s 192.168.1.100 -p 5600 --soft-id 4 --json
```

## 常用参数

| 参数 | 说明 |
|---|---|
| `-s, --server` | 服务端 IP |
| `-p, --port` | 服务端端口 |
| `--soft-id` | 软件 ID |
| `--product-info` | 信息位掩码 |
| `-m, --mode` | `session` 或 `long` |
| `-c, --config` | 配置文件 |
| `-i, --interactive` | 交互模式 |

## 机器人应该怎么问

- 服务端地址是多少
- 连接方式是命令行还是程序内调用
- 是否能直接用客户端连通
- 返回里 `usedNodes` 和 `maxUsers` 是多少

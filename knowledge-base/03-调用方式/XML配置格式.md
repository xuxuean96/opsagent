# XML 配置格式

## 机器人需要检查的内容

- XML 声明编码是否正确
- 路径是否存在
- 节点名称是否和组件一致
- 属性映射是否完整
- 处理顺序是否合理

## 常见问题

- 编码不一致
- 路径分隔符错误
- 节点命名不匹配
- 属性映射漏配
- 处理配置顺序冲突

## 建议回答方式

先让用户贴：

- XML 配置片段
- 输入文件路径
- 输出文件路径
- 对应报错日志

# 组件总览

## 位置

- `DDBExec.exe` / `DDBExec_64.exe`
- `DDBDispatch.dll`
- `DDBCore.dll`
- `DataPsr.dll`
- `DDBDplPsr.dll`
- 各 CAD/Office 驱动 DLL

## 机器人先识别什么

1. 组件类型
2. 数据源类型
3. 操作模式
4. 系统位数
5. 日志文件

## 关键调用链

- eCol: COM -> DDBExec -> XML -> 驱动 DLL -> XML/缓存
- PLM-CLOUD: Java -> 命令行 -> DDBExec -> 驱动 DLL -> XML/缓存
- 测试桩: GUI/CLI -> XML 配置 -> DDB 执行

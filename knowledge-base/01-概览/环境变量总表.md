# 环境变量总表

| 变量 | 作用 | 重点检查 |
|---|---|---|
| `KMSoftPath` | 公共组件路径 | 是否包含 DDB 公共目录 |
| `PRO_COMM_MSG_EXE` | Creo 通信程序 | 是否指向正确安装目录 |
| `UGII_ROOT_DIR` | UG/NX 路径 | 版本和位数是否匹配 |
| `ACAD` | 字体与 DWG 资源路径 | 是否能找到字体文件 |
| `UGII_UTF8_MODE` | UG 编码模式 | 是否按要求开启 |

## 机器人提问建议

- 你的系统是 Windows 还是 Linux
- 组件是 32 位还是 64 位
- 环境变量是否已经配置
- 组件日志有没有报路径找不到
